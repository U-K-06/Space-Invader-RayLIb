# Space Defender Clone

A simple **Defender / Space Invader–style game** made using **.NET** and **Raylib**, created to **practice and learn Raylib**.

---

## Features

- Player can move left and right.
- Shoot bullets to destroy enemies.
- Enemies spawn in rows and move across the screen.
- Basic scoring system.
- Simple 2D graphics using PNG sprites.

---

## Technologies Used

- **Language:** C#  
- **Framework:** .NET  
- **Graphics Library:** [Raylib](https://www.raylib.com/)  

---

## How to Run

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/space-defender.git
    ```
2. Navigate to the project folder:
    ```bash
    cd space-defender
    ```
3. Build and run the project:
    ```bash
    dotnet run
    ```

---

## Project Structure
- **`obj/`** — Build files (ignored)  
- **`bin/`** — Build output (ignored)  
- **`RevenantFight.mp3`** — Background music  
- **`game.png`** — Player sprite  
- **`space-invader.png`** — Enemy sprite  


---

## Notes

- This project was created for **learning purposes** and is a **practice game**.  
- Collisions are handled simply using **X/Y coordinates**.  
- Designed to be easily extendable for more features like multiple enemy types, power-ups, etc.
